# CareerFlow initial wireframe

## Visual direction

- Calm, professional SaaS interface.
- Deep navy background with white surfaces and emerald accents.
- Large headings, generous spacing, and restrained animation.
- Rounded cards, thin borders, and accessible contrast.

## Mobile behavior

- Navigation collapses into a menu.
- Dashboard sidebar becomes a bottom navigation bar.
- Cards stack in a single column.
- Tables and boards use horizontal scrolling only when unavoidable.
