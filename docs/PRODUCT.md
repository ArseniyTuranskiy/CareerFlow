# CareerFlow product outline

## Purpose

CareerFlow helps a user organize a job search and long-term career development
without switching between spreadsheets, notes, and task managers.

## Main user

A developer or other specialist who is looking for a job, improving skills, and
wants to see measurable career progress.

## Core pages

1. **Landing page** — product explanation and sign-up call to action.
2. **Dashboard** — weekly progress, upcoming tasks, and recent applications.
3. **Applications** — a board with Saved, Applied, Interview, Offer, and Rejected stages.
4. **Goals** — career goals with milestones and deadlines.
5. **Tasks** — focused daily and weekly actions.
6. **Skills** — skill progress and learning plans.
7. **Analytics** — application conversion and productivity trends.
8. **Profile and settings** — personal information and preferences.

## First release

The first usable release will include the landing page, dashboard, and a local
job-application board. Authentication and a database will be added only after
the interface and data model are clear.
